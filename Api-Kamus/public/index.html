English Dictionary API

Free API: Simple-English-Dictionary API is and always will be free. Our mission is to provide users with an API that they can use our provided dictionary data and also contribute their own dictionary data by bringing us your query. This English Dictionary API can be used for free without any API key. You do not need to register or use a credit-card.

Endpoints:

/api/:word : This will return a definition on passed word param. /api/:word1/:word2 : This will return a definition 2 words passed as param :word1 and :word2. /download/all/words : This will return wordlist of all the words available in the API If the passed word doesn't exist in API the it will return following error object: 

{
  err: "word {passedWord} doesn't exist in dictionary" 
}

How it works: The basic syntax of a URL to make a get request to the API is as follows: 
/api/:word
Or you can obtain two-word meanings with a single get query like: 
/api/:word1/:word2
/api/
Modify the Input to check for other words too. 

          
          

            Purpose:  There was
            only one free API dictionary on the market that was down for some
            time, then I thought there should be another free one as well. So
            after looking up a lot of dictionary data from various sources and
            trying to make an API with it and failing again and again. I finally
            succeeded in building this API using a json data which you can also
            find on
            this GitHub repository.
          

          

            Limitations:  Each IP address
            has a limit of 400 queries every 10 minutes. 

            This limit may be raised in the future if I receive contributions
            for more words to this API. 

            Note: 
             The API was created woth JSON file data, and while most words are defined, you may come across incomplete information or missing definitions. If a word is not found, the API will return an err object. To access the full wordlist, simply click on the Download Wordlist button. You can assist us with your queries and help in improving and expanding the API's data by contacting us via the contact form at the bottom of the page.
